# Curiocity

React Native mobile app built with Expo. Features authentication screens with social login support (Apple, Google, Facebook, Twitter).

## Setup

```bash
git clone https://github.com/erictang0220/Curiocity-FrontEnd
cd Curiocity-FrontEnd
npm install
expo start
```

## Tech Stack

- React Native (Expo)
- React Navigation (stack navigator)
- Social auth screens: Apple, Google, Facebook, Twitter
